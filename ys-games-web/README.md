# منصة ألعاب YS Games - دليل الاستضافة والتشغيل

هذا المجلد يحتوي على ملفات الواجهة البرمجية لموقع الألعاب الخاص بك والذي سيتم ربطه بتطبيق الأندرويد.

## الملفات المتاحة:
1. `index.html`: الصفحة الرئيسية للمنصة (بوابة الألعاب)، مصممة بشكل عصري ومظلم وتدعم تشغيل الألعاب بملء الشاشة مع زر رجوع عائم.
2. `games.json`: ملف التكوين الذي يحتوي على قائمة الألعاب. يمكنك إضافة أي لعبة جديدة هنا وسيتم تحديثها تلقائياً داخل تطبيق الأندرويد دون الحاجة لتحديث الـ APK!

---

## طريقة الاستضافة على Vercel:

1. قم برفع هذه الملفات (`index.html` و `games.json`) بالإضافة إلى ملف اللعبة الخاص بك (`car-game.html`) إلى مستودع GitHub الخاص بموقعك.
2. قم بربط المستودع بموقعك على **Vercel** لكي يتم رفعها على الرابط `https://ys-games.vercel.app`.
3. تأكد من أن الرابط النهائي للعبة هو:
   `https://ys-games.vercel.app/car-game.html`
   وأن رابط ملف التكوين هو:
   `https://ys-games.vercel.app/games.json`

---

## طريقة إضافة لعبة جديدة في المستقبل:

كل ما عليك فعله لإضافة لعبة جديدة هو فتح ملف `games.json` وإضافة كائن جديد للعبة داخل المصفوفة بالشكل التالي:

```json
  {
    "id": "new-game-id",
    "title": "اسم اللعبة الجديدة",
    "description": "وصف قصير للعبة يظهر للمستخدم في قائمة الألعاب.",
    "url": "https://ys-games.vercel.app/new-game.html",
    "icon": "🎮"
  }
```

بمجرد حفظ الملف ورفعه على Vercel، ستظهر اللعبة الجديدة فوراً لجميع المستخدمين الذين قاموا بتثبيت تطبيق الأندرويد!
